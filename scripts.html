Scripts¶

Installation of mapmuts installs a set of scripts that can be run directly from the command line. These scripts perform all of the main functions of the package. You should be able to utilize the full functionality simply by running these scripts, without interacting directly with the Python code.

Recommended scripts¶

Here are scripts that are recommended as part of the current suggested workflow for inferring amino-acid preferences.

  • mapmuts_makealignments.py
  • mapmuts_alignmentsummaryplot.py
  • mapmuts_parsecounts.py
  • mapmuts_parsesummaryplots.py
  • mapmuts_countparsedmuts.py
  • mapmuts_inferpreferences.py
  • mapmuts_preferencescorrelate.py
  • mapmuts_preferencemeans.py
  • mapmuts_siteprofileplots.py
  • mapmuts_entropycomparison.py

Other scripts¶

Here are some other scripts that are part of the package. These scripts are not part of the recommended workflow as they have been replaced by other scripts or seem to be unnecessary. However, they are still distributed with the package and might be of use for some application.

  • mapmuts_aafracsplots.py
  • mapmuts_inferenrichment.py
  • mapmuts_enrichmentcorrelate.py
  • mapmuts_enrichmentgeomeans.py
